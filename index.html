This demo shows how to dynamically load different memes into the trading card's template. Click on the buttons above to see how it works!

Eventually these are going to be deployed on the Ethereum blockchain as ERC721 tokens which means that the cards can be owned and there will be a limited amount of each card in existence.

Check out the source code and talk to the rest of the team in our Discord. If you happen to be experienced in Solidity and know about the ERC-721 standard you're welcome to join the dev team!
